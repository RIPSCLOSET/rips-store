⚡ Instant Publish — one-time setup

Do this once and “PUBLISH TO SITE”, “MARK SOLD”, and “DELETE” all work with one tap — no copy-pasting ever again. It uses a free GitHub “token” (like a spare key to your repo that only works from this page).

How to get your token (2 minutes)
  1. On GitHub (logged in), go to github.com/settings/personal-access-tokens → Generate new token.
  2. Name it rips-admin. Expiration: 90 days (you'll make a fresh one when it expires).
  3. Under Repository access choose Only select repositories → pick your store repo.
  4. Under Permissions → Repository permissions find Contents → set to Read and write.
  5. Tap Generate token, copy the long code that starts with github_pat_, paste it below.

🔐 The token is saved only in this browser on this phone/computer. Your partner does the same setup once on their device with their own token.
